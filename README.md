San3a - Flutter Graduation Project

Follow these steps to get the app running on your local machine.

## 🚀 Prerequisites

Make sure you have the following installed:
* [Flutter SDK](https://docs.flutter.dev/get-started/install) (Latest Stable Version)
* [Android Studio](https://developer.android.com/studio) (for Android Emulator) or **Xcode** (for iOS Simulator)
* [VS Code](https://code.visualstudio.com/) with the **Flutter** and **Dart** extensions.

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/AhmedRzk/san3a.git
    ```
2.  **Navigate into the project folder:**
    ```bash
    cd san3a
    ```
3.  **Install dependencies:**
    Run this command in your terminal to download all the required packages listed in `pubspec.yaml`:
    ```bash
    flutter pub get
    ```

## How to Run

1.  Open the project in **VS Code**.
2.  Ensure a simulator is running or a physical device is connected (check the bottom right corner of VS Code).
3.  Press `F5` or run the following command in the terminal:
    ```bash
    flutter run
    ```

## Project Structure
* `lib/` - Contains all the Dart code and logic.
* `assets/` - Images, fonts, and static files.
* `pubspec.yaml` - Project configuration and dependencies.
